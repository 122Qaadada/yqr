# Gradient Poster Visual Design

## Goal

Apply the reference image's black-and-white gradient poster style across the portfolio while preserving the existing content and Earth Hero video.

## Approved Direction

- Use direction A: full-site image-poster treatment.
- Keep the 4K seamless Earth Hero video as the homepage background.
- Add a hard-edged light-beam composition, heavy film grain, monochrome contrast, and restrained cyan accents.
- Make project and capability sections feel like editorial poster panels rather than generic template cards.
- Preserve readability, navigation, contact actions, and the existing page structure.

## Visual Rules

- Dominant palette: black, charcoal, graphite, silver, off-white.
- Accent palette: small amounts of cyan only for navigation, metadata, and focus points.
- Shapes: hard edges, rectangular light gates, projector-like cones, minimal rounded corners.
- Texture: visible film grain and subtle scan structure.
- Layout: left-side editorial copy against large atmospheric negative space.

## Verification

- `pnpm test` must pass, including checks for poster-style CSS tokens.
- `pnpm run build` must pass.
- Browser preview must render without console errors.
